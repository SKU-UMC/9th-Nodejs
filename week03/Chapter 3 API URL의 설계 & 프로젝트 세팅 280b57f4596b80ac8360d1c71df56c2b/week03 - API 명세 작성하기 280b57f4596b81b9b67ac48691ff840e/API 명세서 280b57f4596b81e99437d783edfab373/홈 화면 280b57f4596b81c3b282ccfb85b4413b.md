# 홈 화면

API Path: /api/v1/home
API Version: v1
선택: GET
설명: 홈 화면에 미션 목록과 유저의 포인트 정보, 미션 수행 횟수가 표시됩니다.

## Request Header

```json
Authorization : Bearer {accessToken}
Content-Type : application/json
```

## Request Body

```sql
{}
// GET요청이므로 X
```
